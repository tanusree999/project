# Selenium Automation Project – SauceDemo

## Project Description

This project automates a simple online shopping flow using Python and Selenium WebDriver
The script logs into the website, sorts products, adds items to the cart, completes checkout, and logs out.

Website used: https://www.saucedemo.com/

## Tools Used

* Python
* Selenium WebDriver
* Chrome Browser
* PyCharm (IDE)

## Features

1. Open the SauceDemo website.
2. Login with valid credentials.
3. Verify the product page title.
4. Sort products by **Price (low to high)**.
5. Add two products to the cart.
6. Open the shopping cart.
7. Complete the checkout process.
8. Verify the order confirmation message.
9. Logout from the application.

## Steps Performed in Automation

1. Launch Chrome browser.
2. Navigate to the SauceDemo website.
3. Enter username and password.
4. Click the login button.
5. Verify the product page title.
6. Sort products using dropdown.
7. Add selected products to the cart.
8. Click the cart icon.
9. Click the checkout button.
10. Enter user information (first name, last name, postal code).
11. Complete the order.
12. Verify the success message.
13. Logout from the application.

## Expected Result

The script should successfully:

* Login to the application
* Add products to cart
* Complete checkout
* Display order confirmation message
* Logout successfully
